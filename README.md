## 🚀 Projects Showcase

The website features my key projects:

- **next-swr-boilerplate**: Simple Next.js Structure for fetching data with SWR hook
- **leben-de-RN-app**: React Native mobile application
- **gjej-makine-al**: Gjej Makine Albania - React Native Expo App
- **traders-championship-components-library**: Components for TradersChampionship Webflow website

- ➰ Framer Motion
